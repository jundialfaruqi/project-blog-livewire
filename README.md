## Cara Penggunaan

Jalankan perintah berikut ini untuk memulai proyek Laravel:

-   `composer install`
-   `php artisan key:generate`
-   `cp .env.example .env`
-   Ubah nama database
-   `php artisan migrate`
-   `php artisan serve`
-   `npm install && npm run dev`
-   Buka `http://localhost:8000` di browser
